## 🚀 クイックスタート

1. リポジトリをクローン：
```bash
git clone https://github.com/yourusername/check_computer_on.git
```

2. ブラウザで `index.html` を開く

## 📝 使用方法

「状態を確認」ボタンをクリックするだけで、コンピュータの電源状態を確認できます。コンピュータが電源オン状態の場合、システムは祝賀メッセージを表示します。
