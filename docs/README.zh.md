## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/yourusername/check_computer_on.git
```

2. 在浏览器中打开 `index.html`

## 📝 使用方法

只需点击"检测状态"按钮即可验证您的电脑开机状态。如果您的电脑处于开机状态，系统将显示庆祝信息。
